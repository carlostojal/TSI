HTML - Hypertext Markup Language. É uma linguagem de descrição de páginas de informação, standard no
WWW
. Com essa linguagem (que, para além do texto, tem comandos para introdução de
imagens, formulários, alteração de fontes,
etc.) podem-se definir páginas que contenham informação nos mais variados formatos:
  • texto
  • som
  • imagens
  • animações
